Feedback Test Form










Note:
Included below are sample GET and POST commands that can be copy/pasted
into your command line or other tool...provided for your convenience!

The 'curl' command to send a GET request to the controller is:

curl -X GET http://localhost:8090/feedback

The 'curl' command to send a POST request to the controller is:

curl -X POST localhost:8090/newFeedback -H "Content-type:application/json" -d "{\"comments\":\"Great service!\",\"rating\":10,\"user\":\"Edward\"}"
